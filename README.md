# Massage Techniques in Shampoo & Conditioning — Interactive Infographic

This package contains a brand-aligned interactive infographic for hairdressing and barbering apprentices.

## Covers
- The 4 massage techniques used in shampooing/conditioning:
  - Effleurage
  - Petrissage
  - Friction
  - Tapotement
- What they are
- How to perform them
- When to use them
- Which hair/scalp types they suit
- What to avoid and why
- A selector tool and quiz

## Files
- `index.html`
- `README.md`

## Publish on GitHub Pages
1. Create a repository.
2. Upload both files.
3. Go to Settings → Pages → Deploy from a branch → `main` → `/ (root)`.
4. Use the Pages URL in Genially with an iframe embed.
